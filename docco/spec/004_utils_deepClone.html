    Свойтсва начинающиеся в $$ не клонируются и не копируются в результат клонирования. Это нужно, чтобы при клонировании не дублировались свойства типа $$list, $$tags, $$doc, хранящие кольцевые ссылки на объекты

      check "do not clone props with $$", ->

    src =

      fld1: fld1 = {name: 'fld1', type: 'integer'}

      $$list: [fld1]

      $$tags:

        tag1: [fld1]

    dest = deepClone src

    expect(dest).sameStructure

      fld1: {name: 'fld1', type: 'integer'}
